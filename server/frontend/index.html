轻松穿透任何内网

tmproxy 是一个基于 WebSocket 的高性能内网穿透工具，支持端口转发、HTTP 代理和 Web 管理界面。

🚀

一键启动

选择系统和架构后下载并启动

下载 
curl -L {{.ServerHTTPURL}}/client?os=linux&arch=amd64 -o tmproxy && chmod +x tmproxy && ./tmproxy client --server {{.ServerWsURL}} --local localhost:8080
📋

快速连接指南

如果您已有二进制文件，直接运行

./tmproxy client \
  --server {{.ServerWsURL}} \
  --local localhost:8080
