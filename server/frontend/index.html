轻松穿透任何内网

tmproxy 是一个基于 WebSocket 的高性能内网穿透工具，支持端口转发、HTTP 代理和 Web 管理界面。

立即开始使用

下载适用于您系统的 tmproxy 客户端，一键开启隧道。

下载 Client 二进制文件

🐧 Linux 一键启动

自动下载、授权并启动客户端：

curl -L {{.ServerHTTPURL}}/client?os=linux -o tmproxy && \
chmod +x tmproxy && \
./tmproxy client --server {{.ServerWsURL}} --local localhost:8080

🪟 Windows PowerShell

在 PowerShell 中复制执行：

Invoke-WebRequest -Uri "{{.ServerHTTPURL}}/client?os=windows" -OutFile "tmproxy.exe";
.\tmproxy.exe client --server "{{.ServerWsURL}}" --local localhost:8080

🚀 快速连接指南

如果您已有二进制文件，直接运行：

./tmproxy client \
  --server {{.ServerWsURL}} \
  --local localhost:8080
